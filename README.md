# An Intelligent Multi-Agent Generative Learning Ecosystem (Edunova)

Edunova is an AI-powered intelligent learning ecosystem that leverages a multi-agent architecture and generative AI to create a personalized, adaptive, and interactive educational experience.

The system is designed to improve learning outcomes by using multiple intelligent agents that collaborate to provide content generation, personalized recommendations, assessments, learning assistance, and student support.

---

## Overview

Edunova aims to enhance education by integrating Artificial Intelligence with a multi-agent ecosystem that adapts to individual learning needs and provides an efficient learning environment.

---

## Features

- Multi-Agent AI Architecture
- Personalized learning recommendations
- Intelligent quiz and assessment generation
- Adaptive learning paths
- AI-based learning assistance
- Student performance analysis
- Generative AI-powered content creation
- Smart educational resource suggestions
- Progress tracking and feedback system

---

## System Architecture

The project follows a collaborative multi-agent ecosystem where different agents perform specific tasks.

### Agent Roles

### Learning Agent
- Understands student requirements
- Recommends learning materials
- Generates customized learning paths

### Content Generation Agent
- Creates educational content
- Generates notes and explanations
- Produces topic summaries

### Assessment Agent
- Generates quizzes and practice questions
- Evaluates student performance
- Provides feedback

### Recommendation Agent
- Suggests relevant learning resources
- Personalizes content based on user behavior

### Analytics Agent
- Tracks progress
- Generates performance insights

## Installation

Clone the repository:

```bash
git clone https://github.com/Harshitha-27-cell/An-Intelligent-Multi-Agent-Generative-Learning-Ecosystem-Edunova-.git
```

Navigate to the project directory:

```bash
cd An-Intelligent-Multi-Agent-Generative-Learning-Ecosystem-Edunova-
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
